# Code Review Skill

Automated code reviews using OpenCode with kimi-k2 AI model.

## Files

- `index.mjs` - CLI tool for code review
- `SKILL.md` - Full documentation
- `package.json` - Package metadata

## Quick Start

```bash
node index.mjs review --repo=owner/repo --pr=1
```
